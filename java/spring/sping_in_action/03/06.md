### Spring profile
>
- 解决了Spring bean要跨各种部署环境的通用问题。在运行时，通过环境相关的bean与当前激活的profile进行匹配，
- Spring能够让相同的部署单元跨多种环境运行，而不需要进行重新构建。
>
