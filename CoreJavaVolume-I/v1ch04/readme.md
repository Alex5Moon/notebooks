#### 4.9 [类设计技巧](https://github.com/lu666666/notebooks/blob/master/CoreJavaVolume-I/v1ch04/09.md)
1. 一定要保证数据私有。
2. 一定要对数据初始化。
3. 不用再类中使用过多的基本类型。
4. 不是所有的域都需要独立的域访问器和域更改器
5. 将职责过多的类进行分解
6. 类名和方法名要能够体现她们的职责。
